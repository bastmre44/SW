Miku y Jake – Una pequeña aventura



Este proyecto es una historia interactiva del tipo “Elige tu propia aventura”, desarrollada únicamente con HTML.



La historia sigue a Miku, un gato observador y prudente, y a Jake, un perro mayor pero lleno de alegría. Un día cualquiera se convierte en algo especial cuando deciden si deben salir más allá de la seguridad de su hogar.



Dependiendo de las decisiones que tome el lector, la historia puede terminar de tres maneras distintas:

un final esperanzador, un día tranquilo o una despedida emotiva.



El proyecto fue desarrollado como práctica de estructura HTML, navegación entre páginas y despliegue local con NGINX.



Tecnologías utilizadas: HTML5, Git y NGINX.

No se utilizó CSS ni JavaScript.
